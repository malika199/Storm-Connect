# Politique de Confidentialité et Protection des Données (RGPD)

## 1. Collecte des Données

Nous collectons les données suivantes :
- Informations d'identification (nom, prénom, email, téléphone)
- Informations de profil (photos, bio, préférences)
- Données de navigation et d'utilisation
- Informations de paiement (gérées par Stripe)

## 2. Utilisation des Données

Les données collectées sont utilisées pour :
- Gérer les comptes utilisateurs
- Faciliter les mises en relation
- Envoyer des notifications aux tuteurs
- Améliorer le service

## 3. Partage des Données

Les données peuvent être partagées avec :
- Les tuteurs désignés par l'utilisateur
- Les prestataires de services (Stripe pour les paiements)
- Les autorités compétentes si requis par la loi

## 4. Conservation des Données

Les données sont conservées pendant la durée d'utilisation du service et conformément aux obligations légales.

## 5. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :
- **Droit d'accès** : Accéder à vos données personnelles
- **Droit de rectification** : Corriger vos données
- **Droit à l'effacement** : Supprimer vos données
- **Droit à la portabilité** : Récupérer vos données
- **Droit d'opposition** : Vous opposer au traitement de vos données

## 6. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données.

## 7. Cookies

La plateforme utilise des cookies pour améliorer l'expérience utilisateur.
